# Cooking Masterclass Catalogue

A responsive Vue 3 single-page catalogue showcasing chef-led courses. Users can browse, see pricing and skill levels, mark favorites to a wishlist, and view a wishlist counter.

## Features
- Course cards with title, chef, price (ZAR), skill level
- Sold Out badges for unavailable courses
- Wishlist with count (duplicates prevented)
- Responsive grid (mobile, tablet, desktop)
- Optional filter: show only available

## Tech
- Vue 3 + Vite
- Local data (no backend)
- Single page app (no router)

## Setup
On the Terminal:
npm install
npm run dev
